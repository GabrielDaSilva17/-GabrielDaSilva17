⚙️ Curiosidade: Como funcionam as APIs do site? (Contato & Feedback)

Para receber mensagens no formulário de contato, eu não uso serviços prontos de terceiros. Eu criei minha própria API RESTful usando Python (Flask). Ela roda de forma robusta e altamente escalável em servidores serverless no Google Cloud Run! A API recebe os dados de forma segura, salva na nuvem em uma planilha de banco de dados do Google Sheets de maneira assíncrona, e me envia um alerta detalhado via e-mail corporativo. Além disso, se por acaso você perder a internet no seu navegador durante o envio, este site possui uma Fila Offline inteligente que guarda sua mensagem em cache local e faz a entrega de forma invisível assim que a conexão se restabelecer.

Já para o formulário de feedback, a integração é feita em tempo real utilizando o Firebase Realtime Database. Os comentários enviados pelos visitantes são salvos instantaneamente e armazenados de forma estruturada e em nuvem no banco de dados do Firebase. Toda essa engenharia de ponta foi desenvolvida por mim para praticar e garantir o máximo desempenho e autonomia! 🚀
